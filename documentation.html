Getting Started

Welcome to the InterviewPro API documentation. Our API enables you to integrate our interview preparation platform into your applications.

Base URL

https://api.interviewpro.com/v1

Quick Start

  1. Sign up for an account
  2. Generate an API key from the dashboard
  3. Make your first API request

Authentication

All API requests require authentication using your API key. Include it in the Authorization header:

Authorization: Bearer YOUR_API_KEY

API Endpoints

Questions

GET /questions
POST /questions
GET /questions/{id}
PUT /questions/{id}
DELETE /questions/{id}

Categories

GET /categories
GET /categories/{id}/questions

Progress

GET /progress
POST /progress
GET /progress/{userId}

Rate Limits

API requests are limited based on your plan:

  • Free: 1,000 requests/hour
  • Pro: 10,000 requests/hour
  • Enterprise: Custom limits

Error Handling

The API uses standard HTTP response codes:

  • 200: Success
  • 400: Bad Request
  • 401: Unauthorized
  • 403: Forbidden
  • 404: Not Found
  • 429: Too Many Requests
  • 500: Server Error

Webhooks

Set up webhooks to receive real-time updates:

POST /webhooks
{
    "url": "https://your-domain.com/webhook",
    "events": ["question.created", "progress.updated"]
}

SDKs & Libraries

Official SDKs are available for:

  • Python: pip install interviewpro
  • JavaScript: npm install @interviewpro/sdk
  • Java: Available via Maven Central
  • Ruby: Available via RubyGems
